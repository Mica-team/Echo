# Echo


Echo is an AI-powered companion robot designed to interact naturally with people through voice, movement, and expressive animations.

## Features
- AI conversations
- Voice input and output
- Robot control
- OLED facial expressions
- ESP32-powered hardware
- Future camera navigation
